# OpenSpec

OpenSpecは、このリポジトリにおける振る舞い変更の正本です。

- 現在の振る舞い仕様は `openspec/specs/` に置く。
- 提案中の変更は `openspec/changes/<change-name>/` に置く。
- 完了した変更は `openspec/changes/archive/` に置く。
- 実装前に `openspec-propose` または `/opsx:propose` を使う。
- 完了前に `openspec validate <change-name>` を実行する。

Superpowersは要件探索、実行計画、TDD、デバッグ、レビューに使い、合意済みの振る舞い契約はOpenSpecへ固定します。
